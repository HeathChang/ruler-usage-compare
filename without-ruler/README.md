# 로그인 페이지 프로젝트

React + TypeScript로 구현된 로그인 페이지입니다.

## 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 빌드
npm run build
```

## 주요 기능

- 이메일, 비밀번호 입력 필드
- 이메일 형식 및 비밀번호 길이 유효성 검사
- 로그인 버튼 클릭 시 API 호출
- 로딩 상태 및 에러 메시지 표시

## 프로젝트 구조

```
src/
  ├── App.tsx          # 메인 앱 컴포넌트
  ├── LoginPage.tsx    # 로그인 페이지 컴포넌트
  ├── api.ts           # API 호출 함수
  ├── main.tsx         # 진입점
  └── *.css            # 스타일 파일
```
## 사용한 프롬프트
```
React + TypeScript로 로그인 페이지를 구현해줘.

요구사항:
- 이메일, 비밀번호 입력 필드
- 간단한 유효성 검사
- 로그인 버튼 클릭 시 API 호출
- 전체 코드는 React 함수 컴포넌트 기준

아직 아키텍처 경험이 많지 않은 개발자라고 생각해줘.
너무 복잡한 구조는 피하고,
일반적인 React 프로젝트에서 자연스럽게 작성해줘.

특별한 아키텍처 규칙이나 폴더 구조 제약은 없어.
```
